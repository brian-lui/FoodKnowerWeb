# Logo Generation Prompts (FoodKnower)

## Size Specs

- Maximum required logo size: `1024x1024 px` (master source asset for app-store marketing icon use).
- Current website header display size: `36x36 px` (`.logo img` in `styles.css`).
- Design legibility target: must remain clear at `48x48 px` and above.
- Preferred delivery: vector (`SVG`) plus PNG exports at `1024`, `512`, `192`, `180`, `48`, and `36` px.
